# 📢 주요 기능
- 로그인/로그아웃
- 글 작성하기, 삭제, 수정, 공유 하기
- 글 클릭 시 조회수 표시
- 토픽 별 포스트 분류
- 조회수가 가장 높은 글이 메인 화면 대표글 지정
- 포스트 리스트 불러오기
- AI를 활용한 자동 글쓰기

<br />

# 😊 협업
- 노션, 디스코드를 이용한 소통, 각자 파트를 분담해 작업
- 매일 작업 완료 후 main브랜치에서 머지 및 팀 노션에 작업 내역 정리

<br />
